# NotebookLM スライド作成用 指示文

## 使い方

### Step 1: NotebookLMにアクセス
https://notebooklm.google.com/

### Step 2: 新規ノートブック作成
「新しいノートブック」をクリック

### Step 3: ソース資料をアップロード
`doc/SEO_LLMO_REPORT.md` をアップロード

### Step 4: スライド生成
1. 画面右側の **「Studio（スタジオ）」** セクションを確認
2. **「スライド資料」** をクリック
3. **鉛筆マーク** をクリックしてカスタム指示を入力
4. 下記の指示文をコピー＆ペースト
5. 形式：**「詳細なスライド」** を選択
6. **「生成」** をクリック

---

## カスタム指示文（以下をコピー）

```
以下の条件でスライド資料を作成してください。

【対象者】
- 70〜80代の経営者
- IT・Webに詳しくない方
- 専門用語を使わずに説明が必要

【スライド構成】
1. 表紙（タイトル：ホームページ検索対策のご報告）
2. 今回のご報告内容（3つのポイントを箇条書き）
3. 検索対策とは何か（Google検索で上位に出る工夫）
4. AI対策とは何か（ChatGPTなどに正しく紹介してもらう工夫）
5. 実施した対策一覧（表形式、6項目程度）
6. 期待できる効果：採用面
7. 期待できる効果：会社の信頼性
8. 以前のホームページとの違い（比較表）
9. 一般的なホームページとの違い（優位性）
10. サイト全体の評価：マーケティング視点
11. サイト全体の評価：採用サイトとしての視点
12. サイト全体の評価：デザイン視点
13. 今後の運用について
14. まとめ

【デザイン・トーン】
- 文字は大きく、1スライドの情報量は少なめ
- 専門用語には必ず（）で補足説明を入れる
- 箇条書きは3〜4項目まで
- 表は縦長より横長を優先
- 色使いは落ち着いたトーン（青系統）
- 「難しそう」と感じさせない親しみやすい表現

【文章スタイル】
- 「〜です」「〜ます」の丁寧語
- 一文は短く（40文字以内目安）
- 数字や具体例を積極的に使用
- 「SEO」は「検索対策」、「LLMO」は「AI対策」と言い換え

【避けること】
- カタカナ専門用語の羅列
- 小さな文字
- 1スライドに情報を詰め込みすぎる
- グラフや複雑な図解
```

---

## 生成後の確認ポイント

- [ ] 文字サイズは十分大きいか
- [ ] 専門用語に補足説明があるか
- [ ] 1スライドの情報量は適切か
- [ ] 全体の流れが論理的か
- [ ] 誤字脱字がないか

---

## 補足：編集が必要な場合

NotebookLMのスライドはPDF出力のため直接編集できません。
編集が必要な場合は以下の方法を検討：

1. **Canva** にPDFをインポートして編集
2. **Google スライド** に画像として貼り付けて上から編集
3. **PowerPoint** でPDFを開いて編集（レイアウト崩れ注意）
